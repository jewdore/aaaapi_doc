## Parameters

### Core Parameters
| Parameter | Type | Required | Default | Range | Description |
|-----|-----|-----|-----|--------|--------------------|
| `image` | file | ✅ Yes | - | - | The image to edit. Must be a valid PNG file, less than 4MB, and square. |
| `prompt` | string | ✅ Yes | - | - | A text description of the desired image edit. The maximum length is 1000 characters for dall-e-2 and 4000 characters for dall-e-3. |
| `model` | string | ❌ No | `dall-e-2` | `dall-e-2`<br />`dall-e-3` | The model to use for image editing. Defaults to dall-e-2. |

### Advanced Parameters
| Parameter | Type | Required | Default | Range | Description |
|-----|-----|-----|-----|--------|--------------------|
| `mask` | file | ❌ No | - | - | An additional image whose fully transparent areas (e.g. where alpha is zero) indicate where image should be edited. Must be a valid PNG file, less than 4MB, and have the same dimensions as image. |
| `n` | integer | ❌ No | `1` | `1-10` | The number of images to generate. Must be between 1 and 10. For dall-e-3, only n=1 is supported. |
| `response_format` | string | ❌ No | `url` | `url`<br />`b64_json` | The format in which generated images are returned. Must be one of url or b64_json. URLs are only valid for 60 minutes after the image has been generated. |
| `size` | string | ❌ No | `1024x1024` | `256x256`<br />`512x512`<br />`1024x1024`<br />`1792x1024`<br />`1024x1792` | The size of the generated images. Must be one of 256x256, 512x512, or 1024x1024 for dall-e-2, and one of 1024x1024, 1792x1024, or 1024x1792 for dall-e-3. |
| `user` | string | ❌ No | - | - | A unique identifier representing your end-user, which can help OpenAI to monitor and detect abuse. |

### Example Request
<CodeGroup>
```curl
curl https://api.openai.com/v1/images/edits \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -F "model=dall-e-2" \
  -F "image=@image.png" \
  -F "prompt=a sunlit indoor lounge area with a pool containing a flamingo" \
  -F "n=1" \
  -F "size=1024x1024"
```

```python
from openai import OpenAI
client = OpenAI()

response = client.images.edit(
  model="dall-e-2",
  image=open("image.png", "rb"),
  mask=open("mask.png", "rb"),
  prompt="A sunlit indoor lounge area with a pool containing a flamingo",
  n=1,
  size="1024x1024"
)

image_url = response.data[0].url
```

```javascript
import OpenAI from "openai";
import fs from "fs/promises";

const client = new OpenAI();

const response = await client.images.edit({
  model: "dall-e-2",
  image: await fs.readFile("image.png"),
  mask: await fs.readFile("mask.png"),
  prompt: "A sunlit indoor lounge area with a pool containing a flamingo",
  n: 1,
  size: "1024x1024"
});

const image_url = response.data[0].url;
```
