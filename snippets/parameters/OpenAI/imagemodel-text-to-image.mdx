## Parameters

### Core Parameters
| Parameter | Type | Required | Default | Range | Description |
|-----|-----|-----|-----|--------|--------------------|
| `prompt` | string | ✅ Yes | - | - | A text description of the desired image(s). The maximum length is 32000 characters for gpt-image-1, 1000 characters for dall-e-2 and 4000 characters for dall-e-3. |
| `model` | string | ✅ Yes | - | - | The model to use for image editing. Defaults to gpt-image-1. |

### Advanced Parameters
| Parameter | Type | Required | Default | Range | Description |
|-----|-----|-----|-----|--------|--------------------|
| `n` | integer or null | ❌ No | `1` | `1-10` | The number of images to generate. Must be between 1 and 10. For gpt-image-1, only n=1 is supported. |
| `quality` | string or null | ❌ No | `auto` | `auto`<br />`high`<br />`medium`<br />`low` | The quality of the image that will be generated. auto (default value) will automatically select the best quality for the given model. high, medium and low are supported for gpt-image-1. |
| `size` | string or null | ❌ No | `auto` | `1024x1024`<br />`1536x1024`<br />`1024x1536`<br />`auto` | The size of the generated images. Must be one of 1024x1024, 1536x1024 (landscape), 1024x1536 (portrait), or auto (default value) for gpt-image-1. |
| `background` | string or null | ❌ No | `auto` | `transparent`<br />`opaque`<br />`auto` | Allows to set transparency for the background of the generated image(s). This parameter is only supported for gpt-image-1. When auto is used, the model will automatically determine the best background for the image. If transparent, the output format needs to support transparency, so it should be set to either png (default value) or webp. |
