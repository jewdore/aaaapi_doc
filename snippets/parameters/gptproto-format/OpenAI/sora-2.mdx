## Parameters
| Parameter | Type | Required | Default | Range | Description |
|-----|-----|-----|-----|--------|--------------------|
| `prompt` | string | ✅ Yes | - | - | Text prompt that describes the video to generate. |
| `image` | file | ❌ No | - | - | Optional image reference that guides generation. |
| `duration` | string | ❌ No | `4` | `4`<br />`8`<br />`12` | Clip duration in seconds. Defaults to 4 seconds. |
| `size` | string | ❌ No | `720x1280` | `720x1280`<br />`1280x720`<br />`1024x792`<br />`1792x1024` | Output resolution formatted as width x height. Defaults to 720x1280. |
