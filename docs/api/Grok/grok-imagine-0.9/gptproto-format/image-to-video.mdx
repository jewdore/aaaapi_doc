Grok's GPTProto format for the image to video API.




<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/api/v3/grok/grok-imagine-0.9/image-to-video" \
  -H "Authorization: sk-***********" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "grok-imagine-0.9",
    "prompt": "跳舞",
    "image": "https://oss.heyoos.com/2025/11/20/4cc8f98d0e7e4b19a5c73b0f8fc8c16c.jpeg",
    "size": ""
}'
```


```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/api/v3/grok/grok-imagine-0.9/image-to-video"

    payload := []byte(`{
"model": "grok-imagine-0.9",
"prompt": "跳舞",
"image": "https://oss.heyoos.com/2025/11/20/4cc8f98d0e7e4b19a5c73b0f8fc8c16c.jpeg",
"size": ""
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "sk-***********")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `Authorization` (Header) | string | ✅ Yes | - | Your API key in the format: `sk-***********` |
| `model` | string | ✅ Yes | `grok-imagine-0.9` | The model identifier |
| `prompt` | string | ✅ Yes | - | The prompt describing the desired video animation |
| `image` | string | ✅ Yes | - | URL of the input image to animate |
| `size` | string | ❌ No | - | Optional size parameter for the output video |
