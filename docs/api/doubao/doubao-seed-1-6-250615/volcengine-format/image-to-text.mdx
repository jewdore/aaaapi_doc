## Overview

This endpoint provides image to text functionality using Volcengine official format.

## Authentication

This endpoint requires authentication using a Bearer token with your API key.

<ParamField header="Authorization" type="string" required>
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="model" type="string" required default="doubao-seed-1-6-250615">
  The endpoint ID for the model. Format: ep-xxxxxxxxxx-xxxxx
</ParamField>

<ParamField body="messages" type="array" required default="[{'role': 'user', 'content': 'Who are you?'}]">
  Array of message objects for the conversation. Each message must have a role (system, user, or assistant) and content.
</ParamField>

<ParamField body="temperature" type="number" default="0.7">
  Controls randomness in the output. Range: 0.0 to 1.0
</ParamField>

<ParamField body="top_p" type="number" default="0.9">
  Nucleus sampling parameter. Range: 0.0 to 1.0
</ParamField>

<ParamField body="max_tokens" type="integer" default="2048">
  The maximum number of tokens to generate
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Whether to stream the response using SSE (Server-Sent Events)
</ParamField>

<ParamField body="frequency_penalty" type="number" default="0.0">
  Penalize frequent tokens. Range: -2.0 to 2.0
</ParamField>

<ParamField body="presence_penalty" type="number" default="0.0">
  Penalize new tokens based on presence. Range: -2.0 to 2.0
</ParamField>

<ParamField body="stop" type="array">
  Up to 4 sequences where the API will stop generating further tokens
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/api/v3/chat/completions" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "model": "doubao-seed-1-6-250615",
  "messages": [
    {
      "role": "user",
      "content": "你好，请介绍一下你自己"
    }
  ],
  "temperature": 0.7,
  "max_tokens": 2048
}'
```

```python Python
import requests
import json

url = "https://gptproto.com/api/v3/chat/completions"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "model": "doubao-seed-1-6-250615",
    "messages": [
        {"role": "user", "content": "你好，请介绍一下你自己"}
    ],
    "temperature": 0.7,
    "max_tokens": 2048
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2, ensure_ascii=False))
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/api/v3/chat/completions"

    payload := []byte(`{
  "model": "doubao-seed-1-6-250615",
  "messages": [
    {
      "role": "user",
      "content": "你好，请介绍一下你自己"
    }
  ],
  "temperature": 0.7,
  "max_tokens": 2048
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
