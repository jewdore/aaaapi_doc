## Overview

Higgsfield's GPTProto format for the generate video API.

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/v1/higgsfield/generate"

    payload := []byte(`{
"prompt": "Generate a dance scene",
"model": "higgsfield-standard",
"motion_id": "5d0eee15-569c-4006-a1c7-46d1c42d9315",
"enhance_prompt": true,
"image": [
"https://heyoo.oss-ap-southeast-1.aliyuncs.com/ai-draw/java/test/022cc0f0-8b08-4e8b-9822-2b41ba6c64ad.png"
],
"seed": 1000,
"height": 1080,
"width": 720
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `prompt` | string | ✅ Yes | `Generate a dance scene` | Prompt parameter |
| `model` | string | ✅ Yes | `higgsfield-standard` | The model to use for the request |
| `motion_id` | string | ❌ No | `5d0eee15-569c-4006-a1c7-46d1c42d9315` | Motion Id parameter |
| `enhance_prompt` | boolean | ❌ No | `true` | Enhance Prompt parameter |
| `image` | array | ❌ No | `[&quot;https://heyoo.oss-ap-southeast-1.aliyuncs.com/ai-draw/java/test/022cc0f0-8b08-4e8b-9822-2b41ba6c64ad.png&quot;]` | Image parameter |
| `seed` | integer | ❌ No | `1000` | Seed parameter |
| `height` | integer | ❌ No | `1080` | Height parameter |
| `width` | integer | ❌ No | `720` | Width parameter |
