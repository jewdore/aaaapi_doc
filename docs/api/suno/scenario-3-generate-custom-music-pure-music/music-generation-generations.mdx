Music generation interface, can generate new music directly or continue existing musicIf you want to continue a user-uploaded music, you need to add `-upload` after the mv field, for example `chirp-v3-5-upload`





<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/suno/generate" \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "prompt": "",
  "tags": "heavy metal",
  "mv": "chirp-v3-5",
  "title": "Starry ",
  "continue_clip_id": null,
  "continue_at": null,
  "infill_start_s": null,
  "infill_end_s": null
}'
```

```python Python
import requests
import json

url = "https://gptproto.com/suno/generate"
headers = {
    "Authorization": "YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
  "prompt": "",
  "tags": "heavy metal",
  "mv": "chirp-v3-5",
  "title": "Starry ",
  "continue_clip_id": null,
  "continue_at": null,
  "infill_start_s": null,
  "infill_end_s": null
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2))
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/suno/generate"

    payload := []byte(`{
"prompt": "",
"tags": "heavy metal",
"mv": "chirp-v3-5",
"title": "Starry ",
"continue_clip_id": null,
"continue_at": null,
"infill_start_s": null,
"infill_end_s": null
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `prompt` | string | ✅ Yes | - | Prompt parameter |
| `tags` | string | ❌ No | `heavy metal` | Tags parameter |
| `mv` | string | ❌ No | `chirp-v3-5` | Mv parameter |
| `title` | string | ❌ No | `Starry ` | Title parameter |
| `continue_clip_id` | string | ❌ No | `None` | Continue Clip Id parameter |
| `continue_at` | string | ❌ No | `None` | Continue At parameter |
| `infill_start_s` | string | ❌ No | `None` | Infill Start S parameter |
| `infill_end_s` | string | ❌ No | `None` | Infill End S parameter |
