## Overview

Alibaba WAN 2.5 is a cutting-edge text-to-image model on Alibaba Cloud’s DashScope. It generates high-quality, detailed images directly from text prompts and supports multiple output resolutions.

## Supported inputs & outputs :

Inputs: Text

Outputs: Text and image

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `prompt` | string | ✅ Yes | `a blue cat` | The positive prompt for the generation. |
| `size` | string | ❌ No | `1024*1024` | 768 ~ 1440 per dimension . The size of the generated image in pixels (width*height). |
| `enable_prompt_expansion` | boolean | ❌ No | `1024*1024` | If set to true, the prompt optimizer will be enabled. |
| `negative_prompt` | string | ❌ No | - | The negative prompt for the generation. |
| `seed` | integer | ❌ No | `-1` | -1 ~ 2147483647. The random seed to use for the generation. -1 means a random seed will be used. |
