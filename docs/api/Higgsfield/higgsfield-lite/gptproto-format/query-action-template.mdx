Query available action templates (motions) for video generation using Higgsfield's GPTProto format API.

<CodeGroup>

```bash cURL
curl --location --request GET 'https://gptproto.com/v1/higgsfield/tpl/motions?size=20&search=' \
--header 'Authorization: YOUR_API_KEY'
```

```python Python
import requests

url = "https://gptproto.com/v1/higgsfield/tpl/motions"
headers = {
    "Authorization": "YOUR_API_KEY"
}

params = {
    "size": 20,
    "search": ""
}

response = requests.get(url, headers=headers, params=params)
result = response.json()
print(result)
```

```javascript JavaScript
const url = "https://gptproto.com/v1/higgsfield/tpl/motions?size=20&search=";
const headers = {
  "Authorization": "YOUR_API_KEY"
};

fetch(url, {
  method: "GET",
  headers: headers
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Error:", error));
```

```go Go
package main

import (
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/v1/higgsfield/tpl/motions?size=20&search="

    req, _ := http.NewRequest("GET", url, nil)
    req.Header.Set("Authorization", "YOUR_API_KEY")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Response

<CodeGroup>

```json 200 - Success
{
  "templates": [
    {
      "id": "template_001",
      "name": "Motion Template 1",
      "description": "Description of motion template"
    }
  ],
  "total": 20
}
```

```json 401 - Invalid signature
{
  "error": {
    "message": "Invalid signature",
    "type": "401"
  }
}
```

```json 403 - Insufficient balance
{
  "error": {
    "message": "Insufficient balance",
    "type": "403"
  }
}
```

```json 500 - Internal server error
{
  "error": {
    "message": "Internal server error",
    "type": "500"
  }
}
```
</CodeGroup>

## Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `size` | integer | ❌ No | `20` | Number of templates to return |
| `search` | string | ❌ No | `""` | Search query to filter templates by name or description |
