## Overview

This endpoint allows you to generate images from text descriptions using GPT Image 1 model.

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required default="sk-***********">
  Your API key in the format: `YOUR_API_KEY`
</ParamField>

<ParamField header="Content-Type" type="string" required default="application/json">
  Must be set to `application/json`
</ParamField>

## Request Body

<ParamField body="prompt" type="string" required default="a cat">
  A text description of the desired image(s). The maximum length is 32000 characters for gpt-image-1.
</ParamField>

<ParamField body="n" type="integer" default="1">
  The number of images to generate. Must be between 1 and 10.
</ParamField>

<ParamField body="background" type="string" default="auto">
  Allows to set transparency for the background of the generated image(s). This parameter is only supported for gpt-image-1. Must be one of `transparent`, `opaque` or `auto` (default value). When `auto` is used, the model will automatically determine the best background for the image.
</ParamField>

<ParamField body="quality" type="string" default="auto">
  The quality of the image that will be generated. Must be one of `high`, `medium`, `low`, or `auto` (default value) for gpt-image-1.
</ParamField>

<ParamField body="size" type="string" default="auto">
  The size of the generated images. Must be one of `1024x1024`, `1536x1024` (landscape), `1024x1536` (portrait), or `auto` (default value) for gpt-image-1.
</ParamField>

<ParamField body="enable_sync_mode" type="boolean" default="false">
  Whether to enable synchronous mode. Options: `false`, `true`
</ParamField>

<ParamField body="response_format" type="string" default="b64_json">
  The format in which the generated images are returned. Must be one of `url` or `b64_json`.
</ParamField>

<CodeGroup>

```bash cURL
curl --location 'https://gptproto.com/api/v3/openai/gpt-image-1/text-to-image' \
  --header 'Authorization: YOUR_API_KEY' \
  --header 'Content-Type: application/json' \

    "size": "1024x1024",

    "response_format": "b64_json"
```


```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/api/v3/openai/gpt-image-1/text-to-image"

    payload := []byte(`{
  "prompt": "a cat",
  "n": 2,
  "background": "auto",
  "quality": "medium",
  "size": "1024x1024",
  "enable_sync_mode": true,
  "response_format": "b64_json"
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
