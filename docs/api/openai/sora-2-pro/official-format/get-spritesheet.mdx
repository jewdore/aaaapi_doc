## sora-2-pro (get spritesheet)

For each completed video, you can also download a **thumbnail** and a ​**spritesheet**​. These are lightweight assets useful for previews, scrubbers, or catalog displays. Use the `variant` query parameter to specify what you want to download. The default is `variant=video` for the MP4.

<CodeGroup>

```bash cURL
curl -L "https://gptproto.com/v1/videos/video_abc123/content?variant=spritesheet" \
  -H "Authorization: YOUR_API_KEY" \
  --output spritesheet.jpg
```

```python Python
import requests

url = "https://gptproto.com/v1/videos/video_abc123/content"
headers = {
    "Authorization": "YOUR_API_KEY"
}
params = {
    "variant": "spritesheet"
}

response = requests.get(url, headers=headers, params=params)

with open("spritesheet.jpg", "wb") as f:
    f.write(response.content)

print("Spritesheet saved to spritesheet.jpg")
```

```javascript JavaScript
const fs = require("fs");

const videoId = "video_abc123";
const url = `https://gptproto.com/v1/videos/${videoId}/content?variant=spritesheet`;
const headers = {
  "Authorization": "YOUR_API_KEY"
};

fetch(url, {
  method: "GET",
  headers: headers
})
  .then(response => response.arrayBuffer())
  .then(buffer => {
    fs.writeFileSync("spritesheet.jpg", Buffer.from(buffer));
    console.log("Spritesheet saved to spritesheet.jpg");
  })
  .catch(error => console.error("Error:", error));
```

```go Go
package main

import (
    "fmt"
    "io"
    "net/http"
    "os"
)

func main() {
    url := "https://gptproto.com/v1/videos/video_abc123/content?variant=spritesheet"

    req, _ := http.NewRequest("GET", url, nil)
    req.Header.Set("Authorization", "YOUR_API_KEY")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    file, _ := os.Create("spritesheet.jpg")
    defer file.Close()

    io.Copy(file, resp.Body)
    fmt.Println("Spritesheet saved to spritesheet.jpg")
}
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `video_id` | string | ✅ Yes | - | The identifier of the video task. |
| `variant` | string | ✅ Yes | `spritesheet` | The type of content to download. Use `spritesheet` for spritesheet image. |
