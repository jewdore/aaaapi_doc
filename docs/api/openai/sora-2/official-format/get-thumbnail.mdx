## Overview

For each completed video, you can also download a **thumbnail** and a ​**spritesheet**​. These are lightweight assets useful for previews, scrubbers, or catalog displays. Use the `variant` query parameter to specify what you want to download. The default is `variant=video` for the MP4.

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required default="sk-***********">
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Parameter
<ParamField path="video_id" type="string" required>
  The identifier of the video task.
</ParamField>

## Request Parameter
<ParamField query="variant" type="string" required default="thumbnail">
  thumbnail for video
</ParamField>

<CodeGroup>
```bash cURL

curl -L "https://gptproto.com/v1/videos/video_abc123/content?variant=thumbnail" \
  -H "Authorization: sk-xxx" \
  --output thumbnail.webp
```
</CodeGroup>
