## Overview

This endpoint provides query result functionality.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `task_id` | string | ❌ No | `sora-2:task_xxxx` | task id : sora-2:task_xxx |




<RequestExample>

```bash cURL
curl -X GET "https://gptproto.com/api/v3/predictions/sora-2:task_xxx/result" \
  -H "Authorization: YOUR_API_KEY" \

```

```python Python
import requests
import json

url = "https://gptproto.com/api/v3/predictions/sora-2:task_xxx/result"
headers = {
    "Authorization": "YOUR_API_KEY"
}


response = requests.get(url, headers=headers)
result = response.json()
print(json.dumps(result, indent=2))
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/api/v3/predictions/sora-2:task_xxx/result"
    req, _ := http.NewRequest("GET", url)
    req.Header.Set("Authorization", "YOUR_API_KEY")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
