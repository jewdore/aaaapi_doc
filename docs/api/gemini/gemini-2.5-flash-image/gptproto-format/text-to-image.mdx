## Overview

Google Gemini 2.5 Flash Image, a powerful new image generation and editing model with advanced features and creative control.

- **Text-to-Image:** Generate high-quality images from simple or complex text descriptions.

## Supported inputs & outputs :

Inputs: Text

Outputs: Text and image

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `model` | string | ✅ Yes | `gemini-2.5-flash-image` | The model to use for the request |
| `prompt` | string | ✅ Yes | `Create a picture of a nano banana dish in a fancy restaurant with a Gemini theme` | The prompt for the generation. |
| `size` | string | ❌ No | `1:1` | 1:1, 3:2, 2:3, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9. default 1:1 | size | Resolution | | :----------- | :--------- | | 1:1 | 1024x1024 | | 2:3 | 832x1248 | | 3:2 | 1248x832 | | 3:4 | 864x1184 | | 4:3 | 1184x864 | | 4:5 | 896x1152 | | 5:4 | 1152x896 | | 9:16 | 768x1344 | | 16:9 | 1344x768 | | 21:9 | 1536x672 | |
