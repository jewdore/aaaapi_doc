Edit and transform images based on text prompts using Google's Gemini 2.5 Flash Image model with GPTProto format API.

<CodeGroup>

```bash cURL
curl --location 'https://gptproto.com/api/v3/google/gemini-2.5-flash-image/image-edit' \
--header 'Authorization: YOUR_API_KEY' \
--header 'Content-Type: application/json' \
--data '{
    "prompt": "The man is reading the book.",
    "aspect_ratio": "1:1",
    "output_format": "jpeg",
    "enable_sync_mode": false,
    "enable_base64_output": false,
    "images": [
        "https://oss.heyoos.com/2025/11/13/8b54e86f21f94731884e92f264e0cc74.jpeg"
    ]
}'
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/api/v3/google/gemini-2.5-flash-image/image-edit"

    payload := []byte(`{
    "prompt": "The man is reading the book.",
    "aspect_ratio": "1:1",
    "output_format": "jpeg",
    "enable_sync_mode": false,
    "enable_base64_output": false,
    "images": [
        "https://oss.heyoos.com/2025/11/13/8b54e86f21f94731884e92f264e0cc74.jpeg"
    ]
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `prompt` | string | ✅ Yes | - | Text description of the desired image transformation or editing instructions |
| `images` | array | ✅ Yes | - | Array of image URLs to be edited. Supports one or multiple input images |
| `aspect_ratio` | string | ❌ No | `"1:1"` | Aspect ratio of the output image. Common values: `"1:1"`, `"16:9"`, `"9:16"` |
| `output_format` | string | ❌ No | `"jpeg"` | Output image format. Supported formats: `"png"`, `"jpeg"`, `"webp"` |
| `enable_sync_mode` | boolean | ❌ No | `false` | Enable synchronous mode. When `true`, the API waits for processing to complete before responding |
| `enable_base64_output` | boolean | ❌ No | `false` | Enable base64 encoded output. When `true`, returns the image as base64 string instead of URL |
