Generate images from text prompts using Google's Gemini 2.5 Flash Image model with GPTProto format API.

<CodeGroup>

```bash cURL
curl --location 'https://gptproto.com/api/v3/google/gemini-2.5-flash-image/text-to-image' \
--header 'Authorization: YOUR_API_KEY' \
--header 'Content-Type: application/json' \
--data '{
    "aspect_ratio": "1:1",
    "enable_base64_output": false,
    "enable_sync_mode": true,
    "output_format": "png",
    "prompt": "The man is reading the book."
}'
```

```python Python
import requests
import json

url = "https://gptproto.com/api/v3/google/gemini-2.5-flash-image/text-to-image"
headers = {
    "Authorization": "YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "aspect_ratio": "1:1",
    "enable_base64_output": False,
    "enable_sync_mode": True,
    "output_format": "png",
    "prompt": "The man is reading the book."
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2))
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/api/v3/google/gemini-2.5-flash-image/text-to-image"

    payload := []byte(`{
    "aspect_ratio": "1:1",
    "enable_base64_output": false,
    "enable_sync_mode": true,
    "output_format": "png",
    "prompt": "The man is reading the book."
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `prompt` | string | ✅ Yes | - | Text description for the image generation |
| `aspect_ratio` | string | ❌ No | `"1:1"` | Aspect ratio of the generated image. Common values: `"1:1"`, `"16:9"`, `"9:16"` |
| `output_format` | string | ❌ No | `"png"` | Output image format. Supported formats: `"png"`, `"jpeg"`, `"webp"` |
| `enable_sync_mode` | boolean | ❌ No | `true` | Enable synchronous mode. When `true`, the API waits for generation to complete before responding |
| `enable_base64_output` | boolean | ❌ No | `false` | Enable base64 encoded output. When `true`, returns the image as base64 string instead of URL |
