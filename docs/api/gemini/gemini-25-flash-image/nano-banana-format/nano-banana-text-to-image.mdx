## Overview

**Nano-Banana Text-to-Image** is Google’s lightweight yet powerful **AI image generation model**, built for creators who need fast, high-quality visuals from simple text prompts. It transforms words into expressive, realistic images with remarkable clarity, composition, and style diversity — all within seconds.

- **Text-to-Image:** Generate high-quality images from simple or complex text descriptions.

## Supported inputs & outputs :

Inputs: Text

Outputs: Text and image

# **Text-to-Image**

<CodeGroup>


## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" default="sk-***********" type="string" required>
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="prompt" default="The cat is climbing on the tree" type="string" required>
  Prompt parameter
</ParamField>

<ParamField body="aspect_ratio" default="1:1" type="string">
  Aspect Ratio parameter  1:1, 3:2, 2:3, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9
</ParamField>

<ParamField body="output_format" default="jpeg" type="string">
  Output Format parameter  png, jpeg  Default png
</ParamField>

<ParamField body="enable_sync_mode" default="false" type="boolean">
  If set to true, the function will wait for the result to be generated and uploaded before returning the response. It allows you to get the result directly in the response. This property is only available through the API.
</ParamField>

<ParamField body="enable_base64_output" default="true" type="boolean">
  If enabled, the output will be encoded into a BASE64 string instead of a URL. This property is only available through the API.
</ParamField>

### Aspect ratio

The model defaults to generates 1:1 squares. The different ratios available and the size of the image generated are listed in this table:

| Aspect ratio | Resolution |
| :----------- | :--------- |
| 1:1          | 1024x1024  |
| 2:3          | 832x1248   |
| 3:2          | 1248x832   |
| 3:4          | 864x1184   |
| 4:3          | 1184x864   |
| 4:5          | 896x1152   |
| 5:4          | 1152x896   |
| 9:16         | 768x1344   |
| 16:9         | 1344x768   |
| 21:9         | 1536x672   |
