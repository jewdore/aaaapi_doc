## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `model` | string | ✅ Yes | `gemini-2.5-flash-image-preview` | The model to use for image generation. Use `gemini-2.5-flash-image-preview` or `gemini-2.5-flash-image` for production workflows. |
| `prompt` | string | ✅ Yes | `A serene landscape with mountains and a lake at sunset` | A text description of the desired image(s). Maximum length: 1000 characters (OpenAI standard) / 32,000 characters (extended support). The more detailed and specific your prompt, the better the results. |
| `n` | integer | ❌ No | `1` | Number of images to generate. Accepts values between 1-10. Each image counts as 1,290 output tokens. |
| `size` | string | ❌ No | `1024x1024` | The size of the generated images. Supported values: - `256x256` - Small square (legacy support) - `512x512` - Medium square - `1024x1024` - Large square (recommended) - `1024x1536` - Portrait orientation - `1536x1024` - Landscape orientation |
| `quality` | string | ❌ No | `standard` | The quality of the image generation. Options: - `standard` - Standard quality, faster generation - `hd` - High definition, more detailed results (if supported) |
| `response_format` | string | ❌ No | `url` | The format in which the generated images are returned. Must be one of: - `url` - Returns HTTPS URLs to the images (valid for 60 minutes) - `b64_json` - Returns base64-encoded JSON data |
| `style` | string | ❌ No | `vivid` | The style of the generated images. Options: - `vivid` - Hyper-real and dramatic images - `natural` - More natural, less hyper-real looking images |
| `user` | string | ❌ No | - | A unique identifier representing your end-user, which can help monitor and detect abuse. |

## OpenAI SDK Compatibility

This endpoint is fully compatible with the OpenAI Python SDK:

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://gptproto.com/v1"
)

response = client.images.generate(
    model="gemini-2.5-flash-image-preview",
    prompt="A serene landscape with mountains and a lake at sunset",
    n=1,
    size="1024x1024",
    quality="hd"
)

image_url = response.data[0].url
print(image_url)
```

## Prompt Engineering Tips

To get the best results from image generation:

### Be Specific and Descriptive
- ✅ Good: "A photorealistic portrait of a tabby cat sitting on a red velvet cushion, soft studio lighting, shallow depth of field"
- ❌ Avoid: "A cat"

### Specify Style and Mood
- Include artistic style: "oil painting", "watercolor", "digital art", "photorealistic", "minimalist"
- Describe atmosphere: "warm sunset lighting", "moody and dramatic", "bright and cheerful"

### Include Technical Details
- Lighting: "golden hour lighting", "studio lighting", "natural sunlight"
- Perspective: "wide-angle shot", "close-up", "bird's eye view"
- Quality: "high detail", "4K quality", "sharp focus"

### Use Composition Keywords
- "centered composition", "rule of thirds", "symmetrical", "dynamic angle"
