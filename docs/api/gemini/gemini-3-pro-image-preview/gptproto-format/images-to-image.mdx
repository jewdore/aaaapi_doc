## Overview

This endpoint allows you to generate images from existing images using Gemini 3 Pro Image Preview model.

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `image` | array | ✅ Yes | N/A | Array of image URLs to use as input for image generation. |
| `prompt` | string | ✅ Yes | `a cat` | A text description of the desired image(s). |
| `size` | string | ❌ No | - | The size of the generated images. Options: `1K`, `2K`, `4K` |
| `aspect_ratio` | string | ❌ No | `1:1` | The aspect ratio of the generated image. Supported values: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `9:16`, `16:9`, `21:9` |
| `n` | integer | ❌ No | `1` | The number of images to generate. Must be between 1 and 10. |
| `background` | string | ❌ No | `auto` | Allows to set transparency for the background of the generated image(s). This parameter is only supported for gpt-image-1. Must be one of `transparent`, `opaque` or `auto` (default value). When `auto` is used, the model will automatically determine the best background for the image. |
| `enable_base64_output` | boolean | ❌ No | `false` | Whether to enable base64 output format. Options: `false`, `true` |
| `enable_sync_mode` | boolean | ❌ No | `false` | Whether to enable synchronous mode. Options: `false`, `true` |
| `output_format` | string | ❌ No | `png` | The output format of the generated image. Example: `png` |

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/api/v3/google/gemini-3-pro-image-preview/image-to-edit"

    payload := []byte(`{
  "image": [
    "https://heyoo.oss-ap-southeast-1.aliyuncs.com/ai-draw/java/test/cto.jpeg",
    "https://heyoo.oss-ap-southeast-1.aliyuncs.com/ai-draw/java/test/cto.jpeg"
  ],
  "aspect_ratio": "1:1",
  "n": 5,
  "enable_base64_output": false,
  "enable_sync_mode": false,
  "output_format": "png",
  "prompt": "Generate close-up photos of characters in the snow",
  "size": "1K"
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
