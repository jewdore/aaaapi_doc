## Overview

This endpoint transforms existing videos based on text descriptions using Google's Gemini Veo 3.1 model. The API accepts a video URI along with a text prompt and returns a long-running operation for video generation.

## Authentication

This endpoint requires authentication using an API key.


<ParamField header="x-goog-api-key" type="string" required default="sk-***********">
  Your API key in the format: `YOUR_API_KEY`
</ParamField>

<ParamField header="Content-Type" type="string" required default="application/json">
  Must be set to `application/json`
</ParamField>


## Parameters

<ParamField path="model_name" type="string" required default="veo-3.1-generate-preview">
  The name of the model to use for the request.
</ParamField>

## Request Body

<ParamField body="instances" type="array" required>
  An array of instances containing the video generation parameters
  <Expandable title="properties">
    <ParamField body="prompt" type="string" required>
      The text prompt describing how to transform the video
    </ParamField>
    <ParamField body="video" type="object" required>
      The source video information
      <Expandable title="properties">
        <ParamField body="uri" type="string" required>
          The URI of the source video file (must be a valid Google Cloud Storage or Generative Language API file URI)
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="parameters" type="object">
  Additional parameters for video generation
  <Expandable title="properties">
    <ParamField body="resolution" type="string">
      The resolution of the output video. Options: `720p`, `1080p`
    </ParamField>
  </Expandable>
</ParamField>
