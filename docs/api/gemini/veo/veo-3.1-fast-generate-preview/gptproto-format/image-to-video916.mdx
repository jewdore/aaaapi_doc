## veo (image to video916)

Gemini's veo 3.1 fast generate preview for the image to video916 API.









<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/v1/veo/videos" \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "prompt": "A beautiful woman is dancing in a European and American style",
  "model": "veo3.1",
  "upsample": true,
  "reference": false,
  "aspect_ratio": "16:9",
  "images": [
    "https://heyoo.oss-ap-southeast-1.aliyuncs.com/20251017_170918_yyy1.jpeg",
    "https://heyoo.oss-ap-southeast-1.aliyuncs.com/20251017_170932_yyy2.jpeg"
  ]
}'
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/v1/veo/videos"

    payload := []byte(`{
"prompt": "A beautiful woman is dancing in a European and American style",
"model": "veo3.1",
"upsample": true,
"reference": false,
"aspect_ratio": "16:9",
"images": [
"https://heyoo.oss-ap-southeast-1.aliyuncs.com/20251017_170918_yyy1.jpeg",
"https://heyoo.oss-ap-southeast-1.aliyuncs.com/20251017_170932_yyy2.jpeg"
]
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `prompt` | string | ✅ Yes | `A beautiful woman is dancing in a European and American style` | Prompt parameter |
| `model` | string | ✅ Yes | `veo3.1` | The model to use for the request |
| `upsample` | boolean | ❌ No | `true` | Upsample parameter |
| `reference` | boolean | ❌ No | `false` | Reference parameter |
| `aspect_ratio` | string | ❌ No | `16:9` | Aspect Ratio parameter |
| `images` | array | ❌ No | `[&quot;https://heyoo.oss-ap-southeast-1.aliyuncs.com/20251017_170918_yyy1.jpeg&quot;, &quot;https://heyoo.oss-ap-southeast-1.aliyuncs.com/20251017_170932_yyy2.jpeg&quot;]` | Images parameter |
