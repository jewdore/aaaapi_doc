## Overview

This endpoint generates videos from images based on text descriptions using Google's Gemini Veo 3.1 model. The API accepts an image URI along with a text prompt and returns a long-running operation for video generation.

## Authentication

This endpoint requires authentication using an API key.

<ParamField header="x-goog-api-key" type="string" required default="sk-***********">
  Your API key in the format: `YOUR_API_KEY`
</ParamField>

<ParamField header="Content-Type" type="string" required default="application/json">
  Must be set to `application/json`
</ParamField>

## Parameters

<ParamField path="model_name" type="string" required default="veo-3.1-fast-generate-preview">
  The name of the model to use for the request.
</ParamField>

## Request Body

<ParamField body="instances" type="array" required>
  An array of instances containing the video generation parameters
  <Expandable title="properties">
    <ParamField body="prompt" type="string" required>
      The text prompt describing the video animation you want to generate from the image
    </ParamField>
    <ParamField body="image" type="object" required>
      The source image information
      <Expandable title="properties">
        <ParamField body="uri" type="string">
          The URI of the source image file (must be a valid Google Cloud Storage or Generative Language API file URI). Either `uri` or `bytesBase64Encoded` must be provided.
        </ParamField>
        <ParamField body="mimeType" type="string">
          The MIME type of the image (e.g., `image/png`, `image/jpeg`). Required when using `bytesBase64Encoded`.
        </ParamField>
        <ParamField body="bytesBase64Encoded" type="string">
          Base64-encoded image data. Either `uri` or `bytesBase64Encoded` must be provided.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="parameters" type="object">
  Additional parameters for video generation
  <Expandable title="properties">
    <ParamField body="durationSeconds" type="integer">
      The duration of the output video in seconds. Default: `8`
    </ParamField>
    <ParamField body="aspectRatio" type="string">
      The aspect ratio of the output video. Options: `9:16`, `16:9`, `1:1`. Default: `9:16`
    </ParamField>
    <ParamField body="resolution" type="string">
      The resolution of the output video. Options: `720p`, `1080p`
    </ParamField>
  </Expandable>
</ParamField>
