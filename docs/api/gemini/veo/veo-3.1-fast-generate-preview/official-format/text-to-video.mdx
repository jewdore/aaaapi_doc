## Overview

This endpoint generates videos from text descriptions using Google's Gemini Veo 3.1 model. The API accepts a text prompt and returns a long-running operation for video generation.

## Parameters


## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `instances` | array | ✅ Yes | - | An array of instances containing the video generation parameters |
| `instances[].prompt` | string | ✅ Yes | `a cat` | The text prompt describing the video content you want to generate |
