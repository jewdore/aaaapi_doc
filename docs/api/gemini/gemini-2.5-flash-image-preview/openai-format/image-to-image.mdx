## Overview

Google Gemini 2.5 Flash Image preview, a powerful new image generation and editing model with advanced features and creative control.

## Supported inputs & outputs :

Inputs: Text and Images

Outputs: Text and image

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" default="sk-***********" type="string" required>
  Your API key in the format: `YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="messages" type="array" required>
  Array of message objects for the conversation

  <Expandable title="properties">
    <ParamField body="role" type="string" required>
      role of user.
    </ParamField>
    <ParamField body="content" type="string" required>
      The prompt for the generation.
    </ParamField>
    <ParamField body="experimental_attachments" type="array" required>
      <Expandable title="properties">
        <ParamField body="contentType" type="string" required>
          Supported MIME types:image/png, image/jpeg, image/webp
        </ParamField>
        <ParamField body="url" type="string" required>
          image url.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="model" default="gemini-2.5-flash-image-preview" type="string" required>
  
</ParamField>

## Image to Image

The default output size for chat mode is 1:1 (1024x1024)

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/v1/chat/completions"

    payload := []byte(`{
"messages": [
{
"role": "user",
"content": "Two people standing together",
"experimental_attachments": [
{
"contentType": "image/png",
"url": "https://mgszhhytxjyitifbalro.supabase.co/storage/v1/object/public/chat-attachments/uploads/guakmauqtr.png"
},
{
"contentType": "image/jpeg",
"url": "https://mgszhhytxjyitifbalro.supabase.co/storage/v1/object/public/chat-attachments/uploads/3ox9ikxbcxc.jpeg"
}
]
}
],
"model": "gemini-2.5-flash-image-preview"
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
