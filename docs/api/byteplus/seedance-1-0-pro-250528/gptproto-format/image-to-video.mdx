## Overview

BytePlus's GPTProto format for the image to video API.









<CodeGroup>

```bash cURL
  -H "Authorization: YOUR_API_KEY" \

  "camerafixed": false,
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/api/v3/contents/generations/tasks"

    payload := []byte(`{
"camerafixed": false,
"content": [
{
"type": "image_url",
"image_url": "https://ark-project.tos-cn-beijing.volces.com/doc_image/seedream_i2i.jpeg"
},
{
"type": "text",
"text": "The camera pans from left to right, revealing the full scene of the image"
}
],
"framepersecond": 24,
"model": "seedance-1-0-pro-250528",
"ratio": "16:9",
"resolution": "1080p",
"seed": -1,
"watermark": false
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `camerafixed` | boolean | ❌ No | `false` | Camerafixed parameter |
| `content` | array | ❌ No | `[{&quot;type&quot;: &quot;image_url&quot;, &quot;image_url&quot;: &quot;https://ark-project.tos-cn-beijing.volces.com/doc_image/seedream_i2i.jpeg&quot;}, {&quot;type&quot;: &quot;text&quot;, &quot;text&quot;: &quot;The camera pans from left to right, revealing the full scene of the image&quot;}]` | Video generation prompt. Can include parameters like --ratio (16:9, 9:16, 1:1), --resolution (480p, 720p, 1080p), --duration (5-10s), --camerafixed (true/false) |
| `framepersecond` | integer | ❌ No | `24` | Framepersecond parameter |
| `model` | string | ✅ Yes | `seedance-1-0-pro-250528` | The model to use for the request |
| `ratio` | string | ❌ No | `16:9` | Ratio parameter |
| `resolution` | string | ❌ No | `1080p` | Resolution parameter |
| `seed` | integer | ❌ No | `-1` | Seed parameter |
| `watermark` | boolean | ❌ No | `false` | Watermark parameter |
