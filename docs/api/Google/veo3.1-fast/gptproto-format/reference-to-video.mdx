package main

import (
    "bytes"

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `prompt` | string | ✅ Yes | - | The positive prompt for the generation. A detailed description of the video content you want to generate. |
| `aspect_ratio` | string | ✅ Yes | `16:9` | The aspect ratio of the generated media. Options: `16:9`, `9:16` |
| `images` | array | ✅ Yes | - | An array of reference image URLs. Maximum 1 image. The reference image will influence the style and content of the generated video. |
| `enhance_prompt` | boolean | ❌ No | `true` | Whether to enhance the prompt automatically. Options: `true`, `false` |

## Usage Notes

- The `images` array can only contain 1 reference image
- Reference images guide the visual style and composition of the generated video
- The image URL must be publicly accessible
- Video generation is asynchronous - use the task ID returned to query status
- Use the [Query Task](docs/api/google/veo3.1-fast/gptproto-format/query-task) endpoint to check generation status
