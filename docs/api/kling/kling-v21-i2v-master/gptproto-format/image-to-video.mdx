## Overview

Kling 2.1 is Kuaishou’s advanced image-to-video generation model released in 2025. Designed for high-fidelity cinematic output, Kling 2.1 transforms reference images — with optional text prompts — into visually rich, lifelike video clips.

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" default="sk-***********" type="string" required>
  Your API key in the format: `YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="prompt" default="The character in the picture walks to the anime garden to find a comb placed on the ground, without changing the proportion of the character'\''s face" type="string" required>
  The positive prompt for the generation.
</ParamField>

<ParamField body="negative_prompt" default="" type="string" >
  The negative prompt for the generation.
</ParamField>

<ParamField body="image" default="https://oss.heyoos.com/ai-draw/user/-ead3cec3-de6b-4ed5-b38c-4d8707731686.png" type="string" required>
    First frame of the video; Supported image formats include.jpg/.jpeg/.png; 
  The image file size cannot exceed 10MB, and the image resolution should not be less than 300*300px.
</ParamField>

<ParamField body="duration" default="5" type="integer">
  The duration of the generated media in seconds. 5, 10.  default 5
</ParamField>

<ParamField body="guidance_scale" default="0.5" type="number">
  The guidance scale to use for the generation. 0.00 ~ 1.00
</ParamField>


```python Python
import requests
import json

url = "https://gptproto.com/api/v3/kwaivgi/kling-v2.1-i2v-master"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {   
    "prompt" : "The character in the picture walks to the anime garden to find a comb placed on the ground, without changing the proportion of the character'\''s face",
    "image": "https://oss.heyoos.com/ai-draw/user/-ead3cec3-de6b-4ed5-b38c-4d8707731686.png", 
    "guidance_scale": 0.5,
    "duration": 5
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2))
```

