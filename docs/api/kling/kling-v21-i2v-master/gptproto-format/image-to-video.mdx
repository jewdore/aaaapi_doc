## Overview

This endpoint provides image to video functionality.

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required default="sk-***********">
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl --location 'https://gptproto.com/api/v3/kwaivgi/kling-v2.1-i2v-master' \
--header 'Content-Type: application/json' \
--header 'Authorization: sk-xxxx' \
--data '{   
    "prompt" : "The character in the picture walks to the anime garden to find a comb placed on the ground, without changing the proportion of the character'\''s face",
    "image": "https://oss.heyoos.com/ai-draw/user/-ead3cec3-de6b-4ed5-b38c-4d8707731686.png", 
    "guidance_scale": 0.5,
    "duration": 5
}'
```

```python Python
import requests
import json

url = "gptproto/api/v3/kwaivgi/kling-v2.1-i2v-master"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {   
    "prompt" : "The character in the picture walks to the anime garden to find a comb placed on the ground, without changing the proportion of the character'\''s face",
    "image": "https://oss.heyoos.com/ai-draw/user/-ead3cec3-de6b-4ed5-b38c-4d8707731686.png", 
    "guidance_scale": 0.5,
    "duration": 5
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2))
```

```javascript JavaScript
const url = "gptproto/api/v3/kwaivgi/kling-v2.1-i2v-master";
const headers = {
  "Authorization": "Bearer YOUR_API_KEY",
  "Content-Type": "application/json"
};

const data = {   
    "prompt" : "The character in the picture walks to the anime garden to find a comb placed on the ground, without changing the proportion of the character'\''s face",
    "image": "https://oss.heyoos.com/ai-draw/user/-ead3cec3-de6b-4ed5-b38c-4d8707731686.png", 
    "guidance_scale": 0.5,
    "duration": 5
};

fetch(url, {
  method: "POST",
  headers: headers,
  body: JSON.stringify(data)
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Error:", error));
```
