## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `model` | string | ✅ Yes | `kling-video-std-10-v1-6` | The model to use for the request |
| `prompt` | string | ✅ Yes | `The plane flew over the head` | Prompt parameter |
| `negative_prompt` | string | ❌ No | - | Negative Prompt parameter |
| `mode` | string | ❌ No | `std` | Mode parameter |
| `aspect_ratio` | string | ❌ No | `16:9` | Aspect Ratio parameter |
| `duration` | string | ❌ No | `10` | Duration parameter |
| `callback_url` | string | ❌ No | - | Callback Url parameter |
